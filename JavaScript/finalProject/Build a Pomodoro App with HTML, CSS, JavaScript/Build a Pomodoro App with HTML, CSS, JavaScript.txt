# Introduction
# What is the Pomodoro Technique?
- The Pomodoro Technique is a time management method developed in the late 80s by Francesco Cirillo.
- The idea is to break down work into short intervals, usually 25 minutes, with short breaks in between sessions, usually 5 to 10 minutes
